# TASK PETS 🐾

Turn your to-do list into an adventure!

TASK PETS is a productivity app where completing real-life tasks earns you experience, levels, and coins. Use your rewards to raise and care for your own virtual pet, making productivity fun and rewarding.

(This app is still very much a work in process, trying to learn react, tailwind css, node.js along the way. I want to add a lot more features and, of course, make the app more polished.)
